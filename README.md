# Diwali Sales Analysis – Python EDA Project

This project performs exploratory data analysis (EDA) on Diwali sales data using Python.  
Skills used include data cleaning, visualization, and generating business insights.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights
- Women customers purchased more than men.
- Adults (26–35 age group) had the highest purchasing power.
- Uttar Pradesh, Maharashtra & Karnataka were the top performing states.
- Food, Clothing & Electronics categories dominated the sales.
